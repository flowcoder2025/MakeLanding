# MakeLanding PRD

## 프로젝트 개요
- **이름**: MakeLanding
- **목표**: 주제를 입력하면 AI가 비디오 배경 + 설득력 있는 카피 + 프리미엄 랜딩 페이지를 자동 생성하는 로컬 도구
- **대상 사용자**: 본인 (추후 SaaS 확장 가능)
- **성공 기준**: 주제 입력 → 5분 내 프리미엄 영상 랜딩 페이지 완성, 레퍼런스 수준의 퀄리티

## 기술 스택
- **언어**: TypeScript
- **프레임워크**: Next.js 14+ (App Router)
- **스타일링**: Tailwind CSS
- **애니메이션**: GSAP / Framer Motion
- **비디오 생성**: ComfyUI (로컬 API)
- **AI 카피**: LLM API (Claude/GPT)
- **테스트**: Vitest

## 레퍼런스 분석 (디자인 스펙)

### 공통 디자인 패턴
| 요소 | 스펙 |
|------|------|
| 레이아웃 | 풀스크린 비디오 배경 + 고정 UI 오버레이 |
| 네비게이션 | 상단 고정 — 로고(좌) + 메뉴(중앙) + CTA버튼(우) |
| 헤드라인 | 좌측 정렬, 대형 볼드 (48~72px), 2~3줄 |
| 강조색 | 브랜드별 포인트 컬러 (빨강/골드/블루 등) |
| CTA | 브랜드 컬러 filled 버튼 + ghost 보조 링크 |
| 서브카피 | 헤드라인 옆 또는 아래, 작은 사이즈 (16~18px) |
| 텍스트 효과 | 타이핑 애니메이션, 글자 단위 fade-in |
| 비디오 | AI 생성 풀스크린 배경, 루프 재생, 음소거 |

### 비디오 스타일 유형
1. **실사 AI 영상**: 사람/장소 중심 시네마틱 영상 (PILSEUNG EDU 스타일)
2. **3D 제품 비주얼**: 제품/재료를 고급스럽게 보여주는 슬로우 카메라 (NOURISH 스타일)
3. **3D 캐릭터 애니메이션**: 스토리텔링형 캐릭터 모션 (AUTONIXLAB 스타일)

## L1: 입력 & AI 기획

### L2: 주제 입력
#### L3: CLI 입력 인터페이스
사용자가 터미널에서 비즈니스 주제를 입력하면 AI가 랜딩 페이지를 기획한다.

##### 태스크 (L4)
1. **WI-001-feat CLI 입력 모듈 구현**
   - 비즈니스명, 업종, 핵심 메시지, 타겟 고객을 입력받는 CLI 인터페이스
   - JSON 설정 파일로도 입력 가능
   - 수용 기준: `npx makelanding generate --config input.json` 실행 시 설정 파싱 완료

### L2: AI 카피라이팅
#### L3: 설득력 있는 카피 자동 생성
LLM을 활용하여 헤드라인, 서브카피, CTA 문구를 자동 생성한다.

##### 태스크 (L4)
2. **WI-002-feat AI 카피 생성 모듈 구현**
   - 입력된 주제에서 헤드라인(2~3줄), 서브카피, CTA 문구 생성
   - 레퍼런스 스타일 반영: 강렬하고 짧은 한글 카피
   - 네비게이션 메뉴 항목도 자동 생성
   - 수용 기준: 주제 입력 시 레퍼런스 수준의 카피 JSON 출력

#### L3: 컬러/스타일 추천
##### 태스크 (L4)
3. **WI-003-feat 브랜드 스타일 자동 결정**
   - 업종/주제에 맞는 포인트 컬러, 배경 톤, 폰트 스타일 결정
   - 비디오 스타일 유형 자동 선택 (실사/3D제품/3D캐릭터)
   - 수용 기준: 스타일 설정 JSON 출력 (컬러 팔레트, 비디오 타입, 폰트)

## 비기능 요구사항
- ComfyUI 로컬 서버 (127.0.0.1:8188) 연동 전제
- Node.js 20+ 환경
- 비디오 파일 웹 최적화 (10MB 이내)

- **ComfyUI**: 로컬 API (비디오/이미지 생성)
- **LLM API**: 카피라이팅 (Claude API 또는 OpenAI API)
- **FFmpeg**: 비디오 후처리
